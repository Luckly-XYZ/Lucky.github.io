项目概述

Lucky 是一个功能丰富的即时通讯生态系统，包含基于 Tauri 和 Vue 3 构建的跨平台桌面客户端、 基于 Spring Boot 3 + Spring Cloud Alibaba 构建的高性能服务端，以及基于 Flutter 开发的移动端应用。

支持一对一聊天、群聊、文件传输、音视频通话、屏幕录制、屏幕截取等核心功能，采用现代化的技术栈， 提供了稳定、高效、安全的即时通讯解决方案。

核心特性

💬 即时通讯

  • 支持文本、表情、图片、文件等多种消息类型
  • 一对一聊天和群组聊天
  • 消息已读回执和撤回功能
  • 消息转发和离线存储
  • 支持富文本格式和链接解析

📞 音视频通话

  • 一对一音视频通话
  • 群组音视频会议
  • 屏幕共享功能
  • 内置噪音抑制
  • 通话录制支持

🔒 安全特性

  • 消息加密传输
  • RSA 非对称加密
  • JWT Token 认证机制
  • 细粒度权限控制
  • 敏感数据加密存储

🌐 跨平台支持

  • 桌面端支持 Windows/macOS/Linux
  • 移动端支持 Android/iOS
  • 统一的服务端架构
  • 响应式设计和适配
  • 多设备消息同步

⚙️ 系统功能

  • 多语言支持 (中/英文)
  • 主题切换 (浅色/深色)
  • 快捷键操作
  • 系统托盘集成
  • 截图和录屏功能

🚀 高性能架构

  • 微服务架构设计
  • 支持万级并发连接
  • 消息队列异步处理
  • 数据库优化和缓存
  • 容器化部署支持
