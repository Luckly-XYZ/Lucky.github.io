项目概述

Lucky 是面向高并发场景的即时通讯平台，包含跨平台桌面客户端、Spring Boot 服务端与 Flutter 移动端。

支持一对一和群聊、文件传输、音视频、消息可靠投递与微服务化部署，关注性能与可运维性。

桌面客户端 服务端 移动端

核心特性

💬 即时消息

文本/图片/文件/位置等类型，支持已读/撤回/离线消息与多设备同步。

📞 音视频

WebRTC 支持一对一与组会话，屏幕共享、通话录制与基本降噪。

🔒 安全

传输层加密、JWT 身份认证与细粒度权限控制，敏感数据加密存储。

⚙️ 系统功能

托盘、快捷键、虚拟列表、高效文件传输与多语言 / 主题支持。

🚀 高性能

微服务、消息队列、Redis 缓存、FTS 搜索与水平扩展能力。

🌐 跨平台

Tauri + Vue 桌面、Flutter 移动、统一协议与 SDK 支持多端接入。

技术栈
Vue 3 Tauri Spring Boot 3 Spring Cloud Alibaba Flutter WebRTC RabbitMQ Redis MinIO
